# CalculoDeRisco

## Visão Geral
CalculoDeRisco é um projeto .NET 8 projetado para calcular vários riscos financeiros usando abordagens padronizadas. Ele inclui módulos para cálculos de capital regulatório, análise de sensibilidade e muito mais.

## Configuração
1. Clone o repositório.
2. Abra a solução no Visual Studio.
3. Compile o projeto para restaurar as dependências.

## Uso
Consulte o arquivo `use_cases_and_manual.txt` para instruções detalhadas e exemplos.
